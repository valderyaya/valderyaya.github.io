編譯指令

rustc xxx.rs 直接編譯

cargo build 用套件管理工具維護專案，編譯好但不會執行

cargo check 不會build，但會判斷會不會CE

cargo run 編譯+執行

變數

宣告變數

let x = 4; //預設不可改變
let mut x = 4; // 可以改變
let x : i64 = 16; // 定義型別
const THREE_HOURS_IN_SECONDS: u32 = 60 * 60 * 3; //宣告常數

變數的可變性

let spaces = "   ";
let spaces = spaces.len();

以上可以編譯

let mut spaces = "   ";
spaces = spaces.len();

以上不能編譯 使用let可以重新產生相同名字的變數，而如果加了mut則不能改變它的型別

tuple的使用

tuple裡面的型別可以不一樣，但内部的元素不能變

fn main() {
    let x: (i32, f64, u8) = (500, 6.4, 1);
    let five_hundred = x.0;
    let six_point_four = x.1;
    let one = x.2;
    let (a, b, c) = (1, 2.0, 3)
}

陣列的使用

陣列的内部元素型別要一樣，陣列的長度不能變

let a: [i32; 5] = [1, 2, 3, 4, 5]; //分號前面是型別， 後面是元素長度

let a = [3; 5]; // 宣告長度為 5 的陣列，元素全部都是3

函式的使用

函式的宣告不用在main上面

fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("測量值爲：{value}{unit_label}");
}

statements 和 expressions

statements 沒有回傳值， expressions會回傳

fn main(){
    let x = 5;
    let y = {
        let x = 3;
        x + 1
    }; // y = 4
}
fn five(x: i32) -> i32 { //回傳值型別
    x + 5 // 不需要寫return，回傳的部分不能加分號
}

if else

判斷式不需要加 ()

fn main() {
    let number = 3;

    if number {
        println!("數字為三");
    } // Error! 判斷式要是bool
}

三元運算子

let number = if condition { 5 } else { 6 };

迴圈

loop{
    // 無窮迴圈
}

從迴圈回傳數值

fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2; // break可以回傳值
        }
    };

    println!("結果為：{result}");
}

loop label

fn main() {
    let mut count = 0;
    'counting_up: loop { // loop label
        println!("count = {count}");
        let mut remaining = 10;

        loop {
            println!("remaining = {remaining}");
            if remaining == 9 {
                break;
            }
            if count == 2 {
                break 'counting_up; // 退出外層迴圈
            }
            remaining -= 1;
        }

        count += 1;
    }
    println!("End count = {count}");
}

for loop in range

fn main() {
    for i in 1..4 {} // i= 1, 2, 3
    for i in 1..=4 {} // i= 1, 2, 3, 4
    for number in (1..4).rev() {
        println!("{number}!");
    }
    println!("升空！！！");
}
